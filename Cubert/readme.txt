Cubert - An intelligent Rubik's Cube solving robot

1. Run com.nxt.Main.java as "leJOS NXT Program"

	- Calibrates sensor for colors (Orientation needed as follows: White facing top, green facing color detector)
		
	- When done you will be asked to press the right button to initialize connection to PC

	- Press the right button

2. Run com.pc.Main.java as "Java Application"

	- Complete scan will be executed

	- Data will be transferred and displayed to/on the pc after NXT program exit (or error)

To SHUTDOWN during execution in a reasonable state (Arm and sensor released)
-> Hold small grey button until Cubert stops

(To kill execution instantly press orange and the small grey button at the same time)
